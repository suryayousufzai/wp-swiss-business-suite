# Changelog

All notable changes to WP Swiss Business Suite will be documented in this file.

## [1.0.0] - 2026-03-01

### Added
- Initial plugin release
- Core plugin architecture with OOP structure
- Database setup with 5 tables (languages, translations, bookings, booking_meta, services)
- Admin interface with menu system
- Dashboard, Bookings, Languages, Services, and Settings pages
- Support for 4 Swiss languages (DE/FR/IT/EN)
- Default language installation (German, French, Italian, English)
- Default services (Consultation, Quick Meeting)
- Plugin activation/deactivation hooks
- GPL v2 license
- Professional README documentation
- Development roadmap

### Coming in Phase 1
- Language switcher widget
- Content translation system
- Booking form shortcode
- Email notifications
- Calendar interface

### Planned for Phase 2
- Invoice generator
- Swiss QR-Bill support
- VAT calculations
- PDF export
- Payment gateway integration
